## About SPC.to

SPC.to est un projet open-source développé avec React et TypeScript. L'objectif est de fournir une interface moderne et intuitive pour consulter vos moyennes EcoleDirecte.


> [!warning]
> This project is not affiliated with EcoleDirecte or Aplim in any way.

## FAQ

- A release date?<br>When it's ready.
- My data?<br>We don't and will never collect any data about you, more details in our privacy policy.
- Do I need an application?<br>No, you don't. Spc.to is a web application therefore you just need a browser, however you can easily install it as an application.

## Contributing

Ecole Directe Plus is built on React. We use VITE development environment to work. Here is how to setup:

```bash
# install dependencies
npm install

# run on localhost
npm run dev

# host the website
npm run host

# run preview of the build
npm run preview
```
